This is a compiled version of `multer@1.4.4`.

Suitable for folks that don't won't to use multer@1.5-lts.1 or later.

Note: only supported on node 12.x and higher.

To use it, add the following to your package.json:

  "multer": "https://github.com/seyoon20087/multer-compiled",


You may also use it in your `resolutions` or `overrides`
